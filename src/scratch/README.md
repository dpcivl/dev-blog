# scratch — 자유 메모장

작업 중 떠오르는 생각을 자유롭게 적어두는 폴더입니다.

## 사용법

- 파일명 자유 (`flutter-routing-thoughts.md`, `대회-아이디어.md`, `오늘의-의문.md` 등)
- 형식 자유 — 완성된 문장 아니어도 OK, 한 줄짜리도 OK
- **이 폴더 내용은 `.gitignored`** — 자동으로 GitHub에 올라가지 않습니다. 사적 메모로 안심하고 쓰세요.

## 포스트로 만들고 싶을 때

Claude에게 명시적으로 지시:

> "scratch/flutter-routing-thoughts.md 정리해서 글로 올려줘"

그러면 Claude가:

1. 그 파일을 읽고
2. inbox 처리와 동일하게 폴리싱 + 이미지 정리 + "공부할 것" 추가 + 보안 스크럽
3. 정식 포스트로 발행 (`src/data/blog/<slug>.md`)
4. 스크래치 파일 최상단에 발행 완료 코멘트 한 줄 추가

## 발행 후

원본 스크래치 파일은 그대로 둡니다. 작성자가 정리하고 싶으면 직접 삭제하거나 다른 곳으로 옮기시면 됩니다.

## 백업 주의

이 폴더는 git에 올라가지 않으므로 **로컬 PC에만 존재**합니다. 중요한 내용이라면 별도 백업 정책(클라우드 드라이브 등) 을 권장합니다. 발행한 포스트는 git에 들어가므로 그건 백업되어 있어요.
